# EU4 Standard Macro Pack — Quick Reference

## Military  `F-row`

| Key | What it does | Before pressing |
|-----|-------------|-----------------|
| **F5** | Split 16k → 8×2k, then siege all | Select army, cursor on province |
| **F6** | Split 16k → 8×2k only | Select army, cursor on province |
| **F7** | Siege this unit (one press) | Select unit, cursor on province |
| **F8** | Siege all visible substacks | Cursor on province |

> **32k army:** tray → Substacks 16, then F6 on each 16k half, then F8.

---

## Diplomacy  `Ctrl + symbol`  *(opens panel automatically)*

| Key | Improve Relations with… |
|-----|------------------------|
| **Ctrl+/** | Neighbouring Countries |
| **Ctrl+.** | Own Subject Countries |
| **Ctrl+,** | Outraged Countries |
| **Ctrl+M** | Allies |
| **Ctrl+N** | Threatening Countries |

> Fires 7 diplomat clicks. Needs a clean screen — closes any open panel first (`b → 0 → a`).

---

## State Edicts  `Ctrl + letter`  *(open panel first)*

Open: `b → 9 → s`  then scroll state list to the rows you want.

| Key | Edict applied to all visible states |
|-----|-------------------------------------|
| **Ctrl+E** | Encourage Development |
| **Ctrl+K** | Age ability edict *(slot 8 in Reformation, changes each age)* |
| **Ctrl+-** | No Edict |

---

## Subjects  `Ctrl+;`  *(open Subjects panel first)*

| Key | Action |
|-----|--------|
| **Ctrl+;** | Enable Divert Trade — subject under cursor *(hover Interactions icon, fire, repeat)* |
| **Ctrl+Shift+;** | Enable Divert Trade — all visible subjects automatically |

---

## Tray Menu  *(right-click taskbar icon)*

| Setting | Options | Default |
|---------|---------|---------|
| Substacks | **8** / 16 | 8 |
| Speed | Fast 25ms / **Normal 50ms** / Slow 100ms | Normal |

A tooltip confirms the change on screen.

---

## Kill switch

**Ctrl+Shift+F12** — terminates the script immediately.
